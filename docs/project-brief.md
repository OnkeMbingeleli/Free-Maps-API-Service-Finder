# Project 03 - Service Finder

## 📋 Original Project Brief

**Overview:** Develop a web application that helps users locate nearby public services such as clinics, libraries, shelters, and more using a Maps API. The app will display services on an interactive map, allowing users to view locations, get directions, and explore relevant details. In the second phase of the project, users will be able to log in and suggest new service locations, which will be stored in a mock database.

---

## 🛠️ Tech Stack Options

### Frontend
- HTML, CSS, JavaScript
- React.js or Vue.js for UI components
- Maps API for displaying locations
- Axios or Fetch for API calls

### Backend
- Node.js with Express for API server
- Firebase Authentication for user login
- Mock database (Firebase Firestore or JSON file)

### Other Tools
- Git for version control
- Postman for API testing
