# Service Finder Blueprint

This repository follows the Service Finder project blueprint.

## Structure

- `backend/` - backend services, auth, integrations, jobs, middleware, config, docs, tests
- `frontend/` - UI, pages, hooks, state, offline support, assets, public, tests
- `shared/` - types and shared constants
- `docs/` - onboarding and workflow documentation
- `.github/` - CI workflows and issue / PR templates

## Notes

- Keep the folder layout aligned with the project brief.
- Add implementation files inside the matching folder as the project grows.
